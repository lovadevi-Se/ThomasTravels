## Thomas Travel -- Mini Project

Thomas Travel is a simple java based mini project that demonstrates the implementation of Object-oriented programming (OOP) concepts.
The project manages travel and driver details using java classes.

## Project Files

--> Driver.java

--> Travel.java

--> TestDriver.java

## Technologies Used

--> Java

--> OOps Concept

This project helps beginners understand how multiple classes interact in java to build a simple travel management system.
